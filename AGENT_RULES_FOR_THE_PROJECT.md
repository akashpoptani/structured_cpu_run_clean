The rule for this project

Codex should never do a giant rewrite.

Every Codex step should be one of these:

"
1. Inspect only
2. Create a new file
3. Copy existing files into structured_cpu_run_clean
4. Add documentation
5. Move one small piece of logic behind a wrapper
6. Add one test/check
7. Run a diff and explain it
"

And every step must end with:

"Do not proceed further until Akash approves."

IMPORTANT : Keep every change small and reviewable. {Less than 20 lines of code}
